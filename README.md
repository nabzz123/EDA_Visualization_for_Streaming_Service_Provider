# Streaming Services Provider – Data Exploration & Visualization
This study focuses on Exploratory Data Analysis (EDA) and visualization techniques to uncover hidden patterns within the streaming platform's content dataset.


## Business Objective
To analyze content trends, release patterns, and audience preferences for a global streaming services provider using exploratory data analysis and visualization techniques.

## Problem Statement
The company aims to understand:
- Content distribution across genres and regions
- Growth patterns in content production
- Viewer-centric insights to optimize content acquisition

## Methodology
- Data Cleaning & Feature Engineering
- Univariate & Bivariate Analysis
- Trend Analysis
- Visualization using Matplotlib & Seaborn

## Key Insights
- Rapid increase in original content post-2016
- Dominance of specific genres across regions
- Seasonal trends in content releases

## Business Impact
Insights can guide:
- Content investment strategy
- Regional market expansion
- Recommendation system enhancements

## Files
- `notebook.ipynb` – Full analysis
- `data/` – Dataset
- `figures/` – Visualizations
